# konukooo - Simple E-commerce Store

konukooo is a modern, full-stack e-commerce web application built with Next.js and Tailwind CSS. It provides a clean and intuitive user experience for browsing products, managing a shopping cart, and simulating the order process. This project is designed to be a robust, resume-worthy showcase of modern web development practices.

## Resume-Ready Project Description

Developed a full-stack e-commerce platform, konukooo, using Next.js, React, and Tailwind CSS. Implemented key features including product browsing, a dynamic shopping cart, and a simulated checkout process. Integrated a GenAI-powered recommendation engine to suggest related products based on item descriptions, enhancing user engagement. Designed a clean, responsive, and accessible user interface with a focus on modern UX principles.

## Tech Stack

*   **Framework**: Next.js (with App Router)
*   **Language**: TypeScript
*   **Styling**: Tailwind CSS & shadcn/ui
*   **State Management**: React Context API
*   **AI**: Genkit for AI flow management

## Features

*   **Product Listing & Details**: Browse a grid of available products and view detailed information for each item.
*   **Shopping Cart**: Add/remove products to the cart and update item quantities. The cart state is managed globally and persists through navigation.
*   **Simulated Checkout**: A streamlined checkout process that demonstrates order placement and cart clearing.
*   **GenAI Product Recommendations**: On each product page, an AI-powered component suggests related items to the user, improving product discovery.
*   **Responsive Design**: The application is fully responsive, providing an optimal experience on devices of all sizes.
*   **Modern UI/UX**: A clean, visually appealing design with a professional color palette, typography, and subtle animations for a smooth user experience.
*   **User Authentication (UI Mock)**: Includes UI pages for user registration and login to demonstrate a complete e-commerce flow.

## Getting Started

### Prerequisites

*   Node.js (v18 or later)
*   npm or yarn

### Setup

1.  Clone the repository:
    ```bash
    git clone <repository_url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd konukooo
    ```
3.  Install the dependencies:
    ```bash
    npm install
    ```

### How to Run

1.  **Run the development server:**
    ```bash
    npm run dev
    ```

2.  Open your browser and navigate to `http://localhost:9002` to see the application in action.
