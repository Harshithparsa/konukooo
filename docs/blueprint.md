# **App Name**: ShopSphere

## Core Features:

- User Authentication: Allow users to register, login, and logout with password hashing and session-based authentication to protect user data and restrict access to protected routes.
- Product Listing & Details: Display products with key information (name, description, price, image, stock) on a product listing page and provide detailed information on individual product pages.
- Shopping Cart Management: Enable users to add products to their cart, remove items, and update quantities.  The cart is stored per logged-in user and displays the total price.
- Order Processing: Provide a checkout page for users to place orders, save order history including user, products, quantity, total price, order date and order status.
- Related product recommendation: Use AI tool to analyze current product's description and recommend related items from database to the user, displayed on product page.

## Style Guidelines:

- Primary color: Deep teal (#2A9D8F) to convey trustworthiness and sophistication, suitable for an e-commerce platform.
- Background color: Light teal (#E9F8F5) to provide a clean and unobtrusive backdrop that complements the primary color.
- Accent color: Coral (#E76F51) to draw attention to key interactive elements such as 'Add to Cart' buttons, creating visual interest and guiding the user's eye.
- Body and headline font: 'Inter' for a modern, neutral and readable design.
- Employ a clean, responsive grid system to ensure the website is visually appealing and functions seamlessly across all devices.
- Use simple, consistent icons to represent common actions and categories, enhancing usability and providing a visual guide to the user.
- Incorporate subtle animations for product loading and cart updates to give the user clear feedback, create a dynamic user experience, and avoid visual fatigue.
